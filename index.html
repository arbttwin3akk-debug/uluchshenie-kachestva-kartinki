Коррекция фотографии

По заданию — не апскейл, а коррекция: яркость, контрастность и цветность (насыщенность). Загрузите фото, подвиньте ползунки — превью обновляется в браузере. Демо не отправляет файл на сервер.

Параметры коррекции

Значения по умолчанию — без изменений (100%). ИИ позже может подбирать оптимальные значения автоматически; здесь — ручная настройка для демонстрации API/интерфейса.

Параметры: яркость 100%, контраст 100%, цветность 100%

«Применить» в демо фиксирует текущие значения в статусе. Экспорт в файл можно добавить через canvas при подключении полного пайплайна.
