Коррекция фотографий

По заданию параметры коррекции определяются автоматически на основе ИИ — яркость, контрастность и цветность подбираются по анализу изображения. Загрузите фото и нажмите «Авто (ИИ)», при необходимости скорректируйте вручную. Превью обновляется в браузере.

Параметры коррекции

По заданию ИИ подбирает оптимальные значения автоматически. Нажмите «Авто (ИИ)» после загрузки — параметры вычисляются по анализу пикселей. Ползунки — для ручной корректировки при необходимости.

Загрузите фото → нажмите для автоподбора
Параметры: яркость 100%, контраст 100%, цветность 100%

ИИ анализирует яркость, контраст и насыщенность по гистограмме и подбирает коррекцию. «Применить» фиксирует результат.
